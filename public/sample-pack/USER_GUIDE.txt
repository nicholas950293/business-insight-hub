Business Insight Hub Sample Pack

Welcome!

This sample pack helps you explore Business Insight Hub without preparing your own dataset.

How to use

1. Open Business Insight Hub.
2. Click "Choose File".
3. Select any CSV file from this sample pack.
4. Wait for the dashboard and analysis to load.

Recommended first file

01_normal.csv

Sample descriptions

01_normal.csv
Clean dataset for first-time users.

02_missing_values_light.csv
Contains a few missing values.

03_missing_values_heavy.csv
Contains many missing values.

04_missing_columns.csv

08_different_products_dataset.csv

09_high_missing_revenue_dataset.csv

10_wrong_date_format_dataset.csv
Contains inconsistent date formats.

Notes

Business Insight Hub automatically detects available columns and only runs compatible analysis modules.

Some analyses may be skipped if required columns are unavailable.

This sample pack is provided for demonstration and portfolio purposes only.
